# Sat_korean
Preprocessing S.A.T. Korean script for deep learning.

## Usage
`https://github.com/deveworld/Sat_korean/` \
Clone this repo.

`cd Sat_korean` \
Cd into repo directory

`pip install -r requirements.txt` \
Install pip modules

`python3 main.py` \
Run the script!

Happy ML!
